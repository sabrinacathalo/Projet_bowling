# Projet_bowling

1- Installer Ubuntu sur VM
2-Installer Bambo et suivre guide 
3- Installer java :
sudo apt install default-jdk
sudo apt install default-jre

4- Lancer bamboo avec la commande
5- Lancer le site: http://localhost:8085
6- S'incrire sur Atlassian
7- Générer une licence: https://my.atlassian.com/products/index?sen=18892559&evalId=18892559&eval=true#license_18892559
8- Installer mysql: sudo apt install mysql-server

9- Installer vscode
10- Configurer git:
git config user.email "MY_NAME@example.com"
git config user.name "FIRST_NAME LAST_NAME"
11- Cloner le projet: https://github.com/sabrinacathalo/Projet_bowling
